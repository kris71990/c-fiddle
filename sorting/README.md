# Sorting Algorithms in C

Implementations of some standard sorting algorithms in C.

`Bubble Sort`
  - Time - O(n^2)
  - Space - O(1)

`Insertion Sort`
  - Time - O(n^2)
  - Space - O(1)

`Selection Sort`
  - Time - O(n^2)
  - Space - O(1)

`Merge Sort`
  - Time - O(n lg(n))
  - Space - O(n)

`Quick Sort`
  - Time - O(n lg(n))
  - Space - O(lg(n))
